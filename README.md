# Saviynt-Chatbot
SaviyntBot is an enterprise IAM tool for Saviynt Identity Cloud. It uses AI to turn natural language into complex SQL for audits, risk, and compliance.  Features NL2SQL: English to SQL for users, roles, and SoD.  Futuristic UI: Glassmorphism, light navy style, and WhatsApp-themed icons.  Secure: Built with React/Vite.
